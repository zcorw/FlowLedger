# Flow‑Ledger × Codex：分里程碑执行手册（可复制执行）

> 用法：把本仓库（或单个 `.prompt.txt`）作为 Codex 输入上下文。每次只投喂**一个**里程碑的 prompt 文件，严格“小步快跑”。

## 里程碑清单（按顺序）
- F0：环境与仓库基线
- C1：货币/汇率模块
- U1：用户与偏好
- B1：Telegram Bot 最小可用
- L1：记账写入主线（消费/存款）
- O1：收据 OCR → 草稿
- Q1：查询/汇总/导出
- S1：定时提醒
- O11：可观测与运维

> 所有金额用 Decimal(20,6)，时间入库 UTC、展示按用户时区；写操作必须提供 `Idempotency-Key`；请求日志透传 `X-Request-Id`。
